### NOTATION ARRAY

## BIG O

# comptage
search => O(n)
access => O(1)
ajouter/supprimer un élément à la fin => O(1)
ex :
- push()
- pop()

ajouter/supprimer un élément au la début => O(n)
ex :
- shift()
- unshift()

# O(n)
- concat()
- splice()
- slice()
- foreach / reduce / filter / map / etc...

# log(n * log n)
- sort()
